    Definition List

    API
    - API is the acronym for Application Programming Interface, which is a software intermediary that allows two applications to talk to each other.
    XML
    - Extensible Markup Language (XML) is a markup language that defines a set of rules for encoding documents in a format that is both human-readable and machine-readable.
    DOM
    - The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content.
    SVG
    - Scalable Vector Graphics (SVG) is an Extensible Markup Language (XML)-based vector image. SVG images and their behaviors are defined in XML text files. This means that they can be searched, indexed, scripted, and compressed.
    DITA
    - The Open Darwin Information Typing Architecture (DITA) is a standard XML-based architecture. It provides architectural features for content modularity, content reuse, and controlled extension of document vocabularies in a way that ensures interoperability of DITA documents.
